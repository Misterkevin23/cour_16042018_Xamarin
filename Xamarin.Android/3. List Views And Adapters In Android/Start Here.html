Aperçu

Cette session présente les ListViews et les Adapters Android. Nous commencerons par les ta^ches élémentaires permettant de remplir une ListView. Puis nous étudierons quelques patterns pour améliorer l'efficacité et quelques techniquespour aider l'utilisateur à nvaiger dans un grand jeu de données. Nous finirons pas démontrer comment répondre à un clic de l'utilisateur sur un élément de la liste.

Objectifs

  1. Remplir une ListView en utilisant un ArrayAdapter
  2. Gérer les événements de click sur un élément de liste
  3. Implémenter un adapter personnalisé
  4. Utiliser le layout recycling et le pattern view-holder
  5. Activer le fast scrolling et coder une section indexer

Exercices

  1. Remplir une ListView en utilisant un ArrayAdapter
    Exercice
    Dans ce premier atelier, vous allez créer un nouveau projet, intégrer les données exemples fournies et utiliser un ArrayAdapter pour afficher les données dans uneListView.
  2. Gérer les événements de click sur un élément de liste
    Exercice
    Ici vous allez vous abonner à l'événement ListView.ItemClick. Quand l'utilisateur clique sur un élément de la liste, vous devrez afficher une boite de dialogue alerte qui affiche l'élément sélectionné.
  3. Implémenter un adapter personnalisé
    Exercice
    Vous allez implémenter un adapter personnalisé qui construit des lignes personnalisées pour une ListView.
  4. Utiliser le layout recycling et le pattern view-holder
    Exercise
    Cette section vous montrera 2 techniques habiteullement utilisées avec une ListView et des adpaters personnalisés pour améliorer l'efficicence layout recycling and view holders.
  5. Activer le fast scrolling et coder une section indexer
    Exercice
    Dans cette partie, vous allez activer deux fonctionnalités du ListView pour aider l'utilisateur à naviguer dans un jeu de données important: fast scrolling et section indexer.
