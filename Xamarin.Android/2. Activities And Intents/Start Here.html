Aperçu

Les Activités sont l'élément d'architecture fondamentale des application Android.Une façon usuelle de décrire une application Android est "Collection d'Activités interdépendantes". Ce cours vous montre comment implémenter cette collaboration entre les Activités. Vous allez voir comment démarrer vos propres activités et comment démarrer des Activités Android. Vous apprendrez également comment passer des arguments à une Activité à son démrrage.

Objectifs

  1. Démarrer une Activité dans votre .apk
  2. Finir une Activité
  3. Passer des arguments à une Activité
  4. Obtenir les résultats d'une Activité
  5. Démarrer une Activité Système
