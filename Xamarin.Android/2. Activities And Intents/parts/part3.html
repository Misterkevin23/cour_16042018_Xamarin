Activities et Intents

Durée

5 minutes

Objectifs

Ce TP vous montre comment mettre fin à une activité par programmation. Ceci est utile lorsque vous devez implémenter un bouton "Enregistrer" ou "Annuler" dans votre interface utilisateur. Vous allez écrire le code pour terminer l'activité "Ajouter un élément" lorsque l'utilisateur clique sur le bouton "Annuler". La ligne rouge dans l'image ci-dessous indique la partie de l'application sur laquelle vous allez travailler.

Challenge

Utilisez les recommendations ci-dessous pour réaliser l'exercice.

  1. Ouvrez AddItemActivity.cs.
  2. Ajoutez du code à la méthode OnCancelClick pour terminer l'Activité.
  3. Exécutez et testez votre application.

Steps

Below are the step-by-step instructions to implement the exercise.

Finish an Activity

  1. Open AddItemActivity.cs.
  2. Locate the OnCancelClick method.
  3. Inside OnCancelClick, add a call to the Activity's Finish method. The code is shown below, hidden behind a button.

    Finish 

    void OnCancelClick(object sender, EventArgs e)
{
  Finish();
}

  4. Run the app to test your work. Navigate to the Add Item Activity. Click the "Cancel" button and verify you return to the Main Activity.

Résumé

Dans cet exercice, vous avez écrit du code pour mettre fin à une activité. Vous n'aurez pas besoin de le faire trés souvent puisque le bouton "Retour" d'Android est généralement suffisant. Cependant, chaque fois que vous voulez fournir une opération explicite "Annuler" ou "Enregistrer", il est utile de savoir comment terminer une activité par programmation.
