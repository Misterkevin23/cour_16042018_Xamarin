Activities et Intents

Durée

5 minutes

Objectifs

L'objectif principal de ce laboratoire est de vous aider à comprendre les objectifs du cours. Vous passerez quelques minutes en utilisant la version complète de l'application que vous allez construire pendant le cours. Cela devrait vous donner un aperçu des modèles d'interaction entre activités que ce cours abordera.

Les exercices pour ce cours vous demandent de remplir une demande de «liste d'épicerie». Toutes les activités, l'ensemble de l'interface utilisateur et la gestion des données les cours vous seront fournis. Votre tâche consistera à implémenter la navigation Activité-Vers-Activité. Dans quelques cas, vous devrez passer des arguments et récolter des résultats.

L'architecture de l'application est illustrée ci-dessous. Chaque capture d'écran est une activité distincte. Les flèches rouges indiquent la navigation activité-vers-activité que vous allez coder. Notez que l'activité "Navigateur" provient de l'installation Android standard et n'est pas fournie avec votre application.

Il n'y a pas de code à écrire dans ce premier exercice. Vous allez simplement exécuter l'application terminée.

Challenge

Utilisez les directives ci-dessous pour réaliser l'exercice.

  1. Ouvrir la GroceryList.sln dans le dossier Part 06 Resources/Completed.
  2. Exécutez l'application GroceryLis
  3. Passez quelques minutes à explorer l'application. Pensez aux activités qui démarrent d'autres activités et aux arguments et valeurs de retour qui sont transmis.
  4. Fermez l'application quand vous avez fini.

Steps

Below are the step-by-step instructions to implement the exercise.

Run the completed application

  1. Open the GroceryList.sln solution in the Part 06 Resources/Completed folder.
  2. Run the GroceryList application.
  3. Touch the ITEMS button to navigate to the Items Activity.
  4. Touch one of the items in the list to navigate to the Details Activity. The Items Activity will pass the item's position in the data model to the Details Activity so it knows which item to display.
  5. Touch the Android Back-button to return to the Items Activity.
  6. Touch the Android Back-button to return to the Main Activity.
  7. Touch the ADD ITEM button to navigate to the Add Item Activity. Enter values for a new item. Touch the SAVE button. The Add Item Activity will send the values for the new item back to the Main Activity.
  8. Touch the ITEMS button to return to the Items Activity. Verify that the new item was added to the list.
  9. Touch the Android Back-Button to return to the Main Activity.
  10. Touch the ABOUT button to navigate to the About Activity.
  11. Touch the LEARN MORE button to launch the web browser. The web browser is a standard Android Activity that your app is using.
  12. Stop the app.
  13. Close the solution.

Résumé

L'application Grocery List montre tous les modèles d'interaction d'activité courants: une activité en commençant une autre, transmettre des arguments, récolter des résultats, et lancer un Activité système. Après avoir passé quelques minutes à travailler avec l'application terminée, vous devriez avoir un bon aperçu des objectifs du reste du cours.
