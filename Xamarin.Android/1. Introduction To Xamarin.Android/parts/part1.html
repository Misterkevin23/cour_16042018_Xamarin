Introduction à Xamarin.Android

Durée

10 minutes

Objectifs

Cet atelier vous permet de créer un proejt Xamarin.Android project. Il n'y a pas de code à écrire; toutefois les instructions vont pointer vers des parties du projet créé.

Etapes

Créer un nouveau projet

  1. Lancer votre IDE (Xamarin Studio or Visual Studio).
  2. Créer un nouveu parojet Xamarin.Android.
    Dans Xamarin Studio, utilisez File > New > Solution, puis choisissez le modèle Android App à partir de la catégorie Android > App.
    Dans Visual Studio, utilisez File > New > Project, puis choisissez le modèle Blank App (Android) à partir de la catégorie Visual C# > Android.
  3. Nommez votre application: TipCalculator.
  4. Choisissez un emplacement pour les fichiers de la solution
  5. Utilisez les valeurs par défaut pour tous les autres paramétres du projet.

Examiner le code source

  1. Exécutez l'application et passez un moment à comprendre son comportement. Le code du modèle possède un bouton unique qui affiche le nombre de fois où il a été cliqué.
  2. Ouvrez le fichier Resource > Layout > Main.axml et afficher sa source.
  3. Examinez le XML qui déclare le Button. En particulier, remarquez que le bouton possède un id qui lui a été assigné.
  4. Ouvrez le fichier MainActivity.cs.
  5. Le modèle génrre une activité pour vous. Les Activités sont les blocks de constructions essentiels des applications Android applications. L'attribut Activity de votre MainActivity a un MainLauncher affecté à true afin de spécifer que cette activité est le point d'entrée de votre application. (i.e. analogue à la méthode Main d'une application Console C#b).
  6. Il y a 2 choses à remarquer dans OnCreate. SetContentView prend un "identifier" pour un fichier XML (Resource.Layout.Main) comme argument et instantie l'UI. FindViewById prend un id et renvoie une référence à la vue qui possède cet id. C'est la façon dont le code accède aux vues de l'UI.

Résumé

Dans cette section vous avez cré un nouveau projet Xamarin.Android.
