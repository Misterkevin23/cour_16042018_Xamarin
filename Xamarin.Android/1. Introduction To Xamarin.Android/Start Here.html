Présentation

Ce cours couvre tous les concepts princcipaux nécessaires pour créer et tester une application simple Xamarin.Android app written. Vous allez créer un nouveau projet, construire une UI en XML, et écrire une logique en code C# pour gérer l'interface. Vous alez également apprendre à utiliser les outils Android pour garder à jour votre environnement.

Objectifs

  1. Créer un projet Xamarin.Android project
  2. Décomposeer une application en Activitébs
  3. Créer l'UI d'une Activité
  4. Ecrire le comportement de l'Activité
  5. Mettre à jour le SDKAndroid

Introduction à Android.

Exercices

  1. Créer un projet Xamarin.Android
    Exercice
    Vous allez commencer par créer un nouveau projet Xamarin.Android dans votre IDE.
  2. Ajouter des vues à un fichier de layout
    Exercice

    Exercise
    Dans cet exercice, vous allez écrire du code C# afin d'accéder aux vuesq de l'UI et réaliser quelques opérations sur les données saisies par l'utilisateur.
  4. MAJ des outils et du SDK
    Exercice
    Finallement, vous allez utiliser le gestionnaire de SDK Android pour metttre à jour votre environnement.
